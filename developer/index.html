                          # 페이보릿미 - 취미여가 큐레이션 플랫폼

                          • 페이보릿미
                          • 2019.07. ~ 2019.10.
                          • 임팩트스테이션
                          • Python, Django, Mysql, import 결제 연동
                          • 개발에 투입되는 경우 개발자가 개발하다 사라진 경우가 많은데 이 프로젝트도 그런 케이스였음
                          • 덕분에 Django 를 써볼수 있었는데 Django 개발 경험이 별로 없는 개발자가 만들던 프로젝트라 마무리하면서 디버깅이 아니라 기본 기능들을 뒤집어 엎고 새로 만들어야 하는 케이스가 많이 발생했다.
                          • 프로젝트 시작부터 제대로 설계해서 Django 프로젝트를 해봐야겠다.
                          • Django 프로젝트의 경우 서버 구동 및 상태 관리를 위해서 gunicorn 을 사용하는데 Node 서버 개발시 활용했던 PM2 를 사용했더니 더 간단하게 깔끔하게 처리가 되었다. 다음 프로젝트를 하게 되도 PM2를 활용하면 좋겠다.
