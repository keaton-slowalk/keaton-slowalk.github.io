# Keaton 프로필

# 개발 이력

  • 개발에만 참여한 경우가 많지 않지만 개발 작업이 포함된 경우 개발 프로젝트로 분류했습니다.

# 프로젝트 매니저 이력

  • 애매하게 참여했던 프로젝트들도 있지만 일단 참여했던 프로젝트 관련 정보를 정리한다는 생각으로 정리합니다.

# About

  • 지난 기록들을 정리해보기 위해서 사이트를 만들었습니다.
