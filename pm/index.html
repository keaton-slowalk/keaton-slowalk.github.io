                                      # 형태소 분석기 개선 연구용역

                                      • 2018.12. ~ 2019.06.
                                      • 조선일보 미디어연구소
                                      • 뉴스트러스트 프로젝트를 하면서 사용한 형태소 분석기 mecab-ko 의 성능 개선 가능성을 연구하기 위한 목적으로 수행한 프로젝트
                                      • mecab-ko 는 기존 세종말뭉치를 기반으로 학습한 내용을 기반으로 형태소 분석 작업을 수행하기 때문에 뉴스 기사 분석에는 최적화되어 있지 않다고 판단해서 뉴스 기사만으로 생성한 말뭉치를 기반으로 학습을 수행한 후에 개선 가능성을 타진해보는 프로젝트
                                      • 연구 용역 수행 결과 뉴스 기반의 말뭉치 학습 작업보다는 뉴스 기사에 많이 들어간 고유 명사가 형태소 분석기 사전에 많이 포함되는 것이 중요하다는 결론.
                                      • 급변하는 트랜드에 맞춰서 새로운 고유 명사들을 추출해서 사전에 지속적으로 업데이트 해주는 작업이 형태소 분석 품질 향상에 중요하다는 결론
